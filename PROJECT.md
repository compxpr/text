# Project Direction

Text is a local-first content utility repository for reusable text assets, notes, snippets, templates, and handoff records.

It should preserve content purpose, audience, source notes, revision status, validation checks, and artifact metadata for reuse across personal, engineering, and documentation workflows.

Shared-suite alignment:

- App Platform: jobs, artifacts, workspaces, handoffs
- Report Studio: document/review packages
- Source Registry: optional source references
- AI Policy: text preservation rules
- Fixture Lab: synthetic content fixtures

Core rule: this repository owns text records, content metadata, validation records, and handoff notes. Shared services add metadata and packaging only.
